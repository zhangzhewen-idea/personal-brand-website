/ · HomePage

数据库 + 展示配置

VideoHero → VideoGallery → MaterialLibrary → MatrixAccounts

/services · ServicesPage

数据库 + 展示配置

CourseHero → CourseGrid / CourseCard → CourseConsultDialog → CourseCTA

course 提供名称、标签、简介、价格、上线状态；时长、课时与特色属于展示配置。

/consulting · ConsultingPage

静态展示配置

ConsultingHero → ServiceGrid / ServiceCard → CooperationWorkflow → CaseStudies

数据库暂无商业服务表，当前全部内容放入 consulting 配置，不伪造 API 实体。

/about · AboutPage

数据库 + 展示配置

AboutHero → CreatorStats → Milestones → BehindTheScenes → ContactSection

basic_info 提供作者资料、统计值、照片、影片与导演列表、联系方式；里程碑属于展示配置。

/login · AuthPage

表单展示状态

BrandPanel → AuthTabs → LoginForm / RegisterForm

登录展示 account、password；注册展示 nickname、account、email、password，confirmPassword 仅为表单校验字段。
